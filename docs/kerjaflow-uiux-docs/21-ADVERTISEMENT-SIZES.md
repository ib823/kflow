| iPhone 5.5" Screenshots | 1242×2208 | PNG/JPG | Min 2, max 10 |

## 3. PRESS KIT ASSETS

### 3.1 Logo Package

| Variant | Format | Sizes |
|---------|--------|-------|
| Primary Logo (Color) | SVG, PNG | 32, 64, 128, 256, 512, 1024 |
| Primary Logo (White) | SVG, PNG | Same as above |
| Primary Logo (Black) | SVG, PNG | Same as above |
| Logo Mark Only | SVG, PNG | 32, 64, 128, 256, 512 |
| Wordmark Only | SVG, PNG | Multiple widths |
| Logo + Tagline | SVG, PNG | Multiple widths |

### 3.2 Logo Usage Guidelines

```
MINIMUM SIZE
- Digital: 32px height minimum
- Print: 10mm height minimum

CLEAR SPACE
- Minimum clear space = height of 'K' in KerjaFlow

DO NOT
- Stretch or distort
- Change colors outside brand palette
- Add effects (drop shadow, glow)
- Place on busy backgrounds
- Rotate
- Outline the logo
```
